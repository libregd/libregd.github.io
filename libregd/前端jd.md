# 基础的前端需求

以下框架任选1个，写页面和组件的前提下，能还原页面，完成基本的数据传输。
- vue
- jQuery
- H5，微信小程序/类似

# 中级的前端要求
- vue、react、Angular 至少1个有比较多的经验
- Webpack，scss，测试 这种生态技能要求比较多
- 对浏览器的dom，api，权限，兼容理解比较多
- 各种编译器都使用踩坑了
- 代码规范

# 高级前端
- 搭建项目能力
- nodejs能力
- 做中后台管理和建设都信手捏来
- 一些新技术，比如可视化，互动技术掌握并有实际产出
- 工程化思维和能力
