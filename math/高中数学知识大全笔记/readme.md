# 主要策略

因为是复习，所以只做书里有的题，从题出发延展，扩展能想到的内容，不以做题为目的，而是发散性的探索关联知识。

- 不考虑阅读者的水平，是写给自己的。
- 采用chatGPT来帮忙答疑，毕竟很多概念自己搜和整理确实比较累，反直觉。而且数学的东西是比较固定的，涉及到概念啥的，物料都比较干净。
- 尽量使用latex语法来写必要的符号。
- 尽量结合python，GNU Octave来做一些综合性扩展。
