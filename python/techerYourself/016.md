输入一行字符，分别统计出其中英文字母，空格，数字和其他字符的个数。

letters，space， digit，others

```python

```

```python
s=input("请输入一个字符串：\n")
letters=0
space=0
digit=0
others=0
for c in s:
    if c.isalpha():
        letters+=1
    elif c.isspace():
        space+=1
    elif c.isdigit():
        digit+=1
    else:
        others+=1

print('char=%d,space=%d,digit=%d,others=%d'%(letters,space,digit,others))

        
```

# 课堂练习

设计程序判断变量的命名是否正确（符合规范）

变量命名的规范：变量名只能包含字母，数字和下划线。变量名可以字母或下划线打头，但不能以数字打头

## 导图

![image-20220624175650680](/Users/zanglie/Library/Application Support/typora-user-images/image-20220624175650680.png)

## 答案

**简单版**

```python
var = input('变量名: ')

if var[0].isalpha() or var[0] == '_':
     for i in var[1:]:
         if not(i.isalnum() or i == '_'):
             print('%s变量名不合法' %var)
             break
     else:
         print('%s变量名合法' %var)
else:
     print('%s变量名不合法' %var)
```
