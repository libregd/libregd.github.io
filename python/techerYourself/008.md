设计程序判断变量的命名是否正确（符合规范）

变量命名的规范：变量名只能包含字母，数字和下划线。变量名可以字母或下划线打头，但不能以数字打头

## 答案

**简单版**

```python
var = input('变量名: ')

if var[0].isalpha() or var[0] == '_':
     for i in var[1:]:
         if not(i.isalnum() or i == '_'):
             print('%s变量名不合法' %var)
             break
     else:
         print('%s变量名合法' %var)
else:
     print('%s变量名不合法' %var)
```
